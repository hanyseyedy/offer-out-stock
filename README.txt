== Description ==

this is a wordpress plugin for display offer that is product out of stock.

Display Box:
  "This product is currently out of stock. we have other suggestions for you".

  Then you can go to relevant category.

== Installation ==

click on install button. then push Activate button. or upload the plugin zip file and install it.
do not need to do anything else. the plugin will be work automatically.

e.g.

1. Upload `offer-out-stock.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates
